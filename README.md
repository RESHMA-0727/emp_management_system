🚀 Employee Management System

A modern Employee Management System built using React and Vite. This application helps manage employee records with a clean, responsive, and user-friendly interface.

✨ Features

- 🔐 Login Authentication
- 📊 Interactive Dashboard
- ➕ Add New Employee
- ✏️ Edit Employee Details
- 👁️ View Employee Information
- 🗑️ Delete Employee
- 🔍 Search Employees
- 🏢 Department Filter
- 💰 Salary Sorting (Low → High / High → Low)
- 💾 Local Storage Support
- 🌙 Modern Dark UI
- 📱 Responsive Design

🛠️ Tech Stack

- React
- Vite
- React Router DOM
- Context API
- JavaScript (ES6+)
- HTML5
- CSS3
- Local Storage

📂 Project Structure

src/
 ├── components/
 ├── context/
 ├── data/
 ├── pages/
 ├── styles/
 ├── App.jsx
 └── main.jsx

▶️ Getting Started

Clone the repository:

git clone https://github.com/RESHMA-0727/emp_management_system.git

Install dependencies:

npm install

Run the development server:

npm run dev
